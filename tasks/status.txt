CopyCatOS Project Status — April 14, 2026
==========================================

COMPLETED FEATURES
===================

Core Shell Components
---------------------

cc-wm — Window Manager
  - Reparenting compositing WM (C + Xlib + MoonRock OpenGL compositor)
  - Snow Leopard title bar with traffic lights (close/minimize/zoom)
  - Drop shadows (Gaussian blur, cached textures, active/inactive)
  - XComposite manual-mode compositing via OpenGL
  - Strut handling (dock/menubar reserve screen space)
  - Edge/corner resize handles with per-zone cursors
  - Smart zoom (toggle saved size / maximized)
  - App grouping (Super+H hides all windows of same WM_CLASS)
  - Keyboard shortcuts: Super+Q/W close, Super+M minimize, Super+Tab cycle
  - Window tracking fix for dock/menubar restart cycles
  - XDamage-based efficient repainting

cc-desktop — Desktop Surface
  - Aurora wallpaper scaled to screen
  - Desktop icons from ~/Desktop with file type detection
  - Grid layout (top-right origin, 90x90 cells, 64x64 icons)
  - White text labels with 1px black drop shadow
  - Selection highlights (rounded rect, Aqua blue)
  - Double-click opens with xdg-open
  - Drag to reposition icons, snap to grid
  - Right-click context menu (New Folder, Sort By, Clean Up, Open Terminal)
  - inotify watch for live file system changes (200ms debounce)

cc-menubar — Global Menu Bar
  - Real menubar_bg.png texture (400x22, tiled horizontally)
  - Scales vertically for configurable height (22-44px)
  - Apple logo with dropdown menu (About, Sleep, Restart, Shut Down, Log Out)
  - Active app name (bold) + menu titles from WM_CLASS tracking
  - Menu dropdowns with selection gradients and separators
  - Real Snow Leopard system tray icons (16 MenuExtras PNGs):
    - WiFi: 5 signal levels from AirPort.menu
    - Volume: 4 levels from Volume.menu
    - Bluetooth: 3 states from Bluetooth.menu
    - Battery: 3 states + percentage from Battery.menu
    - Spotlight icon, Clock (Pango text)
  - Configurable height via ~/.config/copycatos/desktop.conf
  - SIGHUP live reload (no restart needed)
  - Hover highlights on menu items

cc-dock — Glass Shelf Dock
  - Pre-composited opaque scurve-xl.png shelf texture
  - Dark 3D ridge ("glass fold") separating back/front shelf zones
  - Bright bottom highlight (RGB 232 — screen edge reflection)
  - Real frontline.png top edge highlight
  - Real separator.png dashed divider between app/doc sections
  - Trapezoid perspective clip (1.4% inset per side)
  - Cosine-falloff icon magnification (3 neighbor range)
  - Two-phase sine bounce animation on app launch
  - Icon reflections (vertically flipped, 50% opacity, fading)
  - Running-app indicator dots (real indicator_medium.png)
  - Drag-and-drop icon reordering
  - Poof removal animation (5-frame sprite sheet)
  - Right-click context menus (Show In Finder, Quit)
  - Stack/folder popups with nine-patch backgrounds
  - Tooltips with 500ms hover delay
  - Process detection every 3 seconds (ps-based)
  - Configurable icon size 32-128px via desktop.conf
  - SIGHUP live reload (no restart needed)
  - All derived sizes scale proportionally from icon_size

cc-spotlight — Search Overlay
  - Ctrl+Space toggle (with NumLock/CapsLock modifier variants)
  - 680px wide rounded rect overlay at 22% from screen top
  - Scans .desktop files from /usr/share/applications + flatpak + local
  - Substring search with name-starts-with priority
  - Max 8 visible results with 32px themed icons
  - Icon cache using djb2 hash map
  - Keyboard navigation (Up/Down/Enter/Escape)
  - Fork + setsid + exec launch

cc-sysprefs — System Preferences App
  - 27 real Snow Leopard pane icons (extracted from .icns via sips)
  - 4 categories: Personal, Hardware, Internet & Wireless, System
  - Toolbar with Show All button + search field
  - Category headers with separator lines
  - Hover highlights on icon cells
  - Click to open pane, Show All to return to grid
  - Functional Dock & Menu Bar pane:
    - Dock icon size slider: 32px (small) to 128px (large)
    - Menubar height slider: 22px (normal) to 44px (large)
    - Interactive draggable knobs (Snow Leopard style)
    - Writes to ~/.config/copycatos/desktop.conf on release
    - Sends SIGHUP to cc-dock and cc-menubar for instant live apply

Qt6 Widget Styling
------------------
  - breeze-aqua (AquaStyle) — Qt6 QStyle plugin (~8,900 lines C++)
  - Snow Leopard Aqua-style push buttons, checkboxes, radio buttons
  - Scrollbars with real Aqua textures (3-piece stretching pattern)
  - Progress bars with Aqua fill gradients
  - Sliders, tabs, text fields
  - Menu rendering, toolbar backgrounds
  - All button states: regular, default (blue), pressed, disabled, inactive

Assets & Infrastructure
-----------------------
  - 2,157 Snow Leopard icons (AquaKDE-icons, freedesktop formatted)
  - 511MB source asset library (snowleopardaura/MacAssets/)
  - 625 runtime assets installed to ~/.local/share/aqua-widgets/
  - Lucida Grande Regular + Bold font
  - 99 original Snow Leopard wallpapers
  - AppImage packaging script template
  - Autologin configured (plasmalogin -> copycatos.desktop)
  - Session script starts all components in correct order
  - Shared config file (~/.config/copycatos/desktop.conf)

SUMMARY
=======

Core desktop: ~75% functional
  - All 6 shell components built and running
  - Real Snow Leopard assets used throughout (no fake gradients)
  - Configurable sizing for handheld (dock 32-128px, menubar 22-44px)
  - Live reload via SIGHUP (no restart needed for size changes)

Main gaps:
  - Finder file manager (biggest missing piece)
  - Window animation polish (genie minimize, traffic light hovers)
  - Functional preference panes beyond Dock/Menubar size
  - Dynamic system status detection (WiFi signal, Bluetooth)
  - Native applications (text editor, terminal, etc.)

Target hardware: Lenovo Legion Go (8.8" 1920x1200, AMD Phoenix1)
Base OS: Nobara 43 (Fedora-based)
Display server: XLibre (X11 fork)
