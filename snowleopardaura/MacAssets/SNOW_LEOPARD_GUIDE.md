# Snow Leopard Asset Guide

This folder contains ~2,000+ original assets extracted from a Mac OS X 10.6 Snow Leopard machine. Use this guide to find and apply assets when recreating the Snow Leopard look in any project.

### SnowReverseOutput — HITheme-Rendered Controls (PRIMARY SOURCE)

These are the most important assets for recreating Aqua. They were generated by
SnowReverse (`/Volumes/Development/SnowReverse/`), a tool that runs on the actual
Snow Leopard machine and uses Apple's own HITheme API to draw every control to a
bitmap. These are **pixel-perfect** — the exact same rendering the OS produces.

**Use these images directly** in AquaExact (personal apps) as 9-slice/resizable
image assets. For AquaDemo (legal recreation), use them as reference to match
gradients, colors, and proportions in code-drawn controls.

## The Aqua Design Language

Snow Leopard is the final refined expression of Aqua before Apple moved to flat design. Key characteristics:

### Color Palette
- **Window title bar (active):** Linear gradient from `#D0D0D0` to `#A0A0A0` with a subtle bright line at the very top edge
- **Window title bar (inactive):** Flatter gradient, `#E8E8E8` to `#D0D0D0`
- **Menu bar:** Semi-translucent white/gray gradient, blurs the wallpaper behind it. See `CoreUI/UI.bundle/Contents/Resources/menubar3.png`
- **Sidebar background:** `#DFE5ED` (blueish gray), slightly darker separator line at the right edge
- **Content area background:** `#FFFFFF` (white) for lists/tables, `#EDEDED` for grouped backgrounds
- **Selection highlight (Aqua):** `#3875D7` to `#5DA0F5` gradient. See `CoreUI/UI.bundle/Contents/Resources/menuselectionaqua.png`
- **Selection highlight (Graphite):** `#808080` to `#A0A0A0` gradient
- **Toolbar background:** Gradient from `#D8D8D8` to `#C0C0C0`, with a 1px separator line at bottom
- **Text colors:** Primary `#000000`, secondary/disabled `#7F7F7F`, inverted `#FFFFFF`

### Window Chrome
- **Brushed metal** was removed in Leopard/Snow Leopard. Windows use unified title/toolbar style.
- **Traffic light buttons** (close/minimize/zoom): 14x14px circular buttons in the title bar. Red `#FF5F57`, Yellow `#FFBD2E`, Green `#28C940` — each with a subtle inner shadow and gloss highlight.
- **Title bar text:** `Lucida Grande 13pt Bold`, centered, with a 1px white drop shadow below the text for embossed effect.
- **Window body:** 1px solid border `#8A8A8A`, with a subtle outer shadow (4px blur, 50% opacity black, 2px y-offset).
- **Resize handle:** Diagonal ridged lines in bottom-right corner. See `CoreUI/UI.bundle/Contents/Resources/resizestandard.png`

### Buttons, Controls, and Widgets
These are stored in `CoreUI/ArtFile.bin` (5.7MB) and `CoreUI/SArtFile.bin` (792KB) as packed binary sprite sheets. To extract individual widget states, use SArtFileTool or ThemePark.

Key characteristics to replicate if you can't extract:
- **Push buttons:** Rounded rect with Aqua gradient — white highlight at top, blue/gray gradient body, subtle inner shadow at bottom. ~20px height for regular, ~17px for small.
- **Default button (blue pulsing):** Same shape, but uses the Aqua blue gradient and originally pulsed gently.
- **Checkboxes:** 14x14px rounded square, Aqua blue fill when checked with white checkmark. See `CoreUI/UI.bundle/Contents/Resources/checkmark.pdf`
- **Radio buttons:** 14x14px circle, Aqua blue fill when selected with centered dot.
- **Text fields:** White background, 1px inset border (darker at top for shadow effect), rounded corners (3px radius).
- **Popup/dropdown buttons:** Same as push buttons but with double arrows on right. See `HIToolbox/popupArrows_black.png` and `popupArrows_white.png`
- **Scrollbars:** Aqua blue rounded thumb on a white/light gray track. Track has subtle inset shadow. Arrows at both ends (top/bottom or left/right).
- **Progress bars:** Animated blue Aqua gradient with horizontal highlight stripes that scroll. See `CoreUI/UI.bundle/Contents/Resources/progressramp.png`, `progressramp2.png`
- **Indeterminate progress:** Animated barber-pole blue/white diagonal stripes.
- **Sliders:** Circular knob with white gloss on a blue-tinted track.
- **Tabs:** Rounded top with subtle gradient, active tab connects to content area. Inactive tabs are slightly recessed.
- **Disclosure triangles:** Small triangles that animate rotation. See `CoreUI/UI.bundle/Contents/Resources/dt_close.pdf`, `dt_open.pdf`, `dt_inbetween*.pdf`

### Materials and Textures
Located in `CoreUI/UI.bundle/Contents/Resources/`:
- `aquamaterial.png` — The signature Aqua blue gradient material
- `brightmaterial.png` — Bright white highlight material
- `glasshighlightmaterial.png` — Glass-like highlight overlay
- `highlightmaterial.png` — General highlight gradient
- `inlaymaterial.png` — Inset/recessed area material
- `trackmaterial.png` — Scrollbar/slider track material
- `metal.png` — Metal/brushed texture (legacy)
- `redmaterial.png`, `greenmaterial.png`, `yellowmaterial.png` — Traffic light button materials
- `bluemap.png`, `graphitemap.png`, `clearmap.png`, `darkbluemap.png` — Color mapping textures for theming controls in Aqua vs Graphite appearance

### Typography
- **System font:** Lucida Grande (Regular 13pt for body, Bold 13pt for emphasis)
- **Small system font:** Lucida Grande 11pt
- **Mini system font:** Lucida Grande 9pt
- **Title bar:** Lucida Grande 13pt Bold, centered, with 1px white shadow
- **Menu items:** Lucida Grande 14pt
- **Labels/captions:** Lucida Grande 10pt in `#7F7F7F`

## SnowReverse Tool

The `SnowReverseOutput/` controls were extracted using SnowReverse, a custom tool
at `/Volumes/Development/SnowReverse/`. It runs on the Snow Leopard machine and
uses Apple's HITheme C API to draw every control type to a CGBitmapContext.

**To re-run or extend:**
1. Edit `SnowReverse.m` to add new control types or sizes
2. Copy to Snow Leopard: `cp SnowReverse.m "/Volumes/Macintosh HD-1/Users/bliz/Desktop/"`
3. On the Snow Leopard machine, run: `cd ~/Desktop && bash build_and_run.sh`
4. Copy results back: `cp -R "/Volumes/Macintosh HD-1/tmp/SnowReverse/" MacAssets/SnowReverseOutput/`

**HITheme functions available** (add more extractions as needed):
- `HIThemeDrawButton` — all button types and states
- `HIThemeDrawTrack` — scrollbars, sliders, progress bars
- `HIThemeDrawTab` / `HIThemeDrawTabPane` — tabs
- `HIThemeDrawFrame` — text fields, list boxes
- `HIThemeDrawMenuBarBackground` / `HIThemeDrawMenuItem` — menus
- `HIThemeDrawSegment` — segmented controls
- `HIThemeDrawGroupBox` / `HIThemeDrawHeader` / `HIThemeDrawPlacard` — containers
- `HIThemeDrawGrowBox` / `HIThemeDrawSeparator` / `HIThemeDrawPopupArrow` — misc

**For exact reproduction (AquaExact / personal apps):**
Use the PNGs from `SnowReverseOutput/` directly as image assets. For buttons and
frames, use 9-slice / resizable image techniques — the center stretches, caps stay fixed.

```swift
// SwiftUI — use extracted button image as resizable
Image("push_active_regular")
    .resizable(capInsets: EdgeInsets(top: 4, leading: 8, bottom: 4, trailing: 8))
    .frame(height: 22)

// AppKit — NSImage with capInsets
let img = NSImage(contentsOfFile: "SnowReverseOutput/buttons/push_active_regular.png")!
img.capInsets = NSEdgeInsets(top: 4, left: 8, bottom: 4, right: 8)
img.resizingMode = .stretch
```

Open the extracted PNGs, zoom in, and sample the exact gradient colors and stop

This gives you the exact colors without using Apple's copyrighted artwork.

## File Format Notes

- **`.icns`** — Multi-resolution icon container. Contains 16x16 through 512x512 sizes. Use `iconutil` to convert: `iconutil --convert iconset MyIcon.icns` creates a folder of PNGs.
- **`.tiff` / `.tif`** — Standard image format, some are multi-page (containing multiple resolutions). Preview.app or `sips` can convert.
- **`.pdf`** — Vector assets (arrows, checkmarks, disclosure triangles). Resolution-independent — preferred for controls that need to scale.
- **`.png`** — Raster assets, often with alpha transparency. Nine-patch style pieces are meant to be tiled/stretched.
- **`.rsrc`** — Classic Mac resource fork files. Contain structured data including images. Use `DeRez` or ResEdit-compatible tools.
- **`.qtz`** — Quartz Composer compositions (screen savers). Not image files — they're animated procedural graphics.

### Converting .icns to PNG
```bash
# Extract all sizes from an .icns file
iconutil --convert iconset /Volumes/Development/MacAssets/Icons/GenericFolderIcon.icns
# Creates GenericFolderIcon.iconset/ with icon_16x16.png, icon_32x32.png, etc.

# Or convert to a single PNG at a specific size using sips
sips -s format png -z 512 512 /Volumes/Development/MacAssets/Icons/GenericFolderIcon.icns --out folder_512.png
```

## Appearance Modes

Snow Leopard supported two appearance modes set in System Preferences > Appearance:

1. **Aqua (Blue)** — Default. Blue selection highlights, blue scrollbar thumbs, blue default buttons. Uses `bluemap.png`, `menuselectionaqua.png`, etc.
2. **Graphite** — Gray selection highlights, gray scrollbar thumbs, gray default buttons. Uses `graphitemap.png`, Graphite variants of assets (e.g., `ecgcl_*` instead of `eccl_*` in Dock stacks, `HiResAppleMenuGraphite.png` instead of `HiResAppleMenu.png`).

When recreating, default to Aqua unless specifically building a Graphite variant.
